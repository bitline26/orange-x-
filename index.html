KYC GUIDE

본인인증, 3분이면 끝!

KYC 인증을 완료해야 입출금, 거래, 이벤트 참여가 가능합니다.

1

Verify Now 클릭

로그인 후 상단의 "Complete Identity Verification" 배너를 클릭하거나, 설정에서 KYC 인증을 시작하세요.

  • 앱 또는 웹에서 모두 가능
  • 로그인 후 바로 안내 배너 표시
2

기본 정보 입력 + 신분증 촬영

이름, 생년월일을 입력하고 신분증(주민등록증/운전면허증/여권 중 택1)을 촬영하여 업로드하세요.

  • First Name / Last Name 영문 입력
  • 신분증 앞뒤 사진 업로드
  • jpg, jpeg, png (5MB 이내)
3

얼굴 인식 (셀피)

카메라 앞에서 얼굴을 3~10초간 보여주면 끝! 업로드 완료 후 "NEXT"를 클릭하세요.

  • 생체 인식으로 본인 확인
  • 승인 완료 시 이메일 안내 발송
  • 미승인 시 재검토 요청 가능
⏱ 평균 소요 시간: 약 3분
